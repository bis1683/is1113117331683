Cloud Services




Infrastructure as a Service:

IaaS is a service which provides the fundamental virtual infrastructures (e.g. servers, firewalls and cloud storage etc.) on demand. The company has full control over the hardware, but the customer can adjust resource consumption based on work load. But the key use of IaaS is in the development and deployment of PaaS, SaaS.

Platform as a Service:

PaaS is a platform which provides the customer with the framework to build and develop a software application. The software built are hosted in the cloud and users can simply access it through a web browser. This provides customer working on the software with secure internet connection and no worries regarding other infrastructural issues (e.g. updates).

Software as a Service:

SaaS is a cloud service which allows the customer to access the software application online at any given time. This is done through the use of a web browser and files saved in the cloud. Instead of buying and installing the software, the customer now pays a subscription monthly or annually, which significantly lowers the cost for updates and the people needed to manage it.

Communication as a Service:

CaaS (a subcategory of Saas) is a cloud technology which provides software integration platform for delivering communications (e.g. Text). It is a middleware software meaning it acts as a bridge between the operating system and applications. Caas is popular among small and median companies as it does not require upfront capital expenses and it is flexible in its feature.
